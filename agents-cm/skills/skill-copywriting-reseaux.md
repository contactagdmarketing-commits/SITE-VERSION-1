## Formats par réseau

### Instagram
- **Post carrousel** : 5-8 slides, slide 1 = accroche choc, slides 2-7 = contenu, slide 8 = CTA
- **Reels** : script de 30-60 secondes, accroche dans les 3 premières secondes
- **Story** : texte court (max 3 lignes), 1 question ou sondage
- **Longueur légende** : 150-300 mots, 5-10 hashtags pertinents en fin
- **Ton** : direct, dynamique, tu tutoie les moins de 25 ans
- **Visuels suggérés** : toujours décrire le visuel idéal en détail pour Canva

### Facebook
- **Post** : 100-250 mots, plus informatif qu'Instagram
- **Ton** : rassurant, professionnel, vouvoiement si contexte parent/adulte
- **Format** : souvent questions ouvertes pour générer des commentaires
- **Pas d'hashtags** (ou max 2-3 très ciblés)
- **Meilleur contenu** : témoignages, conseils pratiques, FAQ, actualités permis

### TikTok
- **Script vidéo** : 15-60 secondes, structure : Accroche (3s) → Problème (5s) → Solution (20s) → CTA (5s)
- **Ton** : très casual, humour accepté, on tutoie systématiquement
- **Accroches qui marchent** : "Tu savais que...", "La vraie raison pour laquelle...", "Stop de croire que..."
- **Toujours finir** par une question pour engager les commentaires
- **Pas de légende longue** — juste l'accroche + 3-5 hashtags

### Google Business
- **Posts** : 150-300 mots, factuel, mots-clés locaux obligatoires
- **Objectif** : être indexé par les IA génératives (Perplexity, ChatGPT, Google AI Overview)
- **Structure** : Contexte local → Service → Preuve sociale → CTA avec numéro
- **Fréquence** : 1-2 posts/semaine minimum pour l'algo Google

## Piliers éditoriaux (thèmes récurrents)

### Pilier 1 : Éducation / Démystification (40% des posts)
Expliquer comment ça marche, lever les peurs, informer sur le candidat libre
Exemples : "Comment passer son permis sans auto-école à Toulouse", "C'est quoi le NEPH ?", "Candidat libre vs auto-école : le vrai comparatif"

### Pilier 2 : Preuve sociale / Témoignages (25% des posts)
Partager des réussites clients (avec permission), avis Google, moments de fierté
Exemples : "Marie a eu son permis du premier coup", "Ce qu'ils disent de Candidalib"

### Pilier 3 : Coulisses / Humanisation (20% des posts)
Montrer la vraie vie de Candidalib, l'accompagnateur, le véhicule, les routes d'examen
Exemples : "Une matinée avec notre accompagnateur", "Le parcours d'examen de Colomiers en détail"

### Pilier 4 : Offre / Conversion (15% des posts)
Posts commerciaux clairs, tarifs, appel à l'action direct
Exemples : "Nos tarifs pour les candidats libres à Toulouse", "Réserve ta première heure d'accompagnement"

## Structure d'un post réussi

```
ACCROCHE (1 ligne qui stoppe le scroll)
   ↓
PROBLÈME ou CONTEXTE (1-2 lignes)
   ↓
DÉVELOPPEMENT / VALEUR (corps du post)
   ↓
CALL TO ACTION (1 action précise)
   ↓
HASHTAGS (si Instagram/TikTok)
```
